# Event-Driven Order Processing

This project demonstrates an **event-driven architecture** for processing orders using Spring Boot and Kafka.  
It handles order creation, validation, and asynchronous processing with separate microservices.  

## Features
- Microservices architecture for order, inventory, and notification handling
- Event-driven communication using Apache Kafka
- PostgreSQL integration for order persistence
- Real-time order status updates and scalable processing

## Technologies
- Java, Spring Boot
- Apache Kafka
- PostgreSQL
- Maven
